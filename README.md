Instrucciones de uso del desarrollo GameTrade:

- Para el uso óptimo de este software hay que tener en cuenta que su respectivo sistema de versiones se encuentra consignado en el enlace del repositorio de Github con este enlace: (Inserta el enlace)
- Para su funcionamiento se debe instalar de antemano el framework .NETcore
- Se debe cambiar la ruta de la base de datos por la correcta en el dispositivo local
- Para su uso se debe usar el gestor de base de datos SQLserver, y se debe crear la base de datos para ver si funcionamiento.
